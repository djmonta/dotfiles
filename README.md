The djmonta's dotfiles
========

## Overview

Personal config files.

## Installation

Run the installation command below in your terminal.

| Tools | The installation command |
|:-:|:-:|
| cURL | `bash -c "$(curl -fsSL dot.djmonta.me)"` |
| Wget | `bash -c "$(wget -qO - dot.djmonta.me)"` |
